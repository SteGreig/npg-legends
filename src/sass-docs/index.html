    settings (colours)

    variables

    Brand

    $Brand: #2e2e2e;

    Description

    Any brand colours which aren't included within the named colours below

    Example

    .foo { color: $IndianRed; background-color: rgba($DeepSkyBlue, .4); }

    Links

    • http://chir.ag/projects/name-that-color/#FFFFFF

    Author

    • Rob Watson

    Social Media

    $Social Media: #3b5998;

    Description

    Social media colours have been named, then mapped to the social network name.

    Example

    .cta--facebook { color: $White; background-color: $Facebook; }

    Author

    • Rob Watson

    Global

    $Global: #CD5C5C;

    Description

    This is a full list of the named colors from the CSS3 specification.

    Example

    .foo { color: $FireBrick; }

    Links

    • https://www.quackit.com/css/color/charts/css_color_names_chart.cfm

    Author

    • Rob Watson

    xs-down

    @mixin xs-down() { ... }

    Description

    It is unlikely that you'll ever use this as you should be building the site with a mobile up approach. Therefore phone portrait should be the default, with styles then overwritten down the cascade using the mixins below.

    $screen-xs-min set inside settings > breakpoints

    Parameters

    None.

    Example

    .foo { @include xs-down; }
