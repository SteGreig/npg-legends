# Accordion

> Useful for FAQ sections

## Configuration 
- Alignment
	- There are 3 options, these being Center, Left & Right
- Theme
	- Returns one of 3 options, this ensures that all of the sections remain on brand.
		- primary
		- secondary
		- alternative

## Options
- Header
- Sub Header
- Copy
- Accordion
	- Panel header
	- Panel content
